# TWS Water Factory Management System

## ระบบจัดการโรงงานผลิตน้ำดื่ม TWS

---

## การติดตั้งและเริ่มใช้งาน

### ข้อกำหนดเบื้องต้น

- **Node.js** เวอร์ชัน 18 ขึ้นไป → ดาวน์โหลดที่ https://nodejs.org

### เริ่มต้นใช้งาน

1. ดับเบิ้ลคลิกที่ไฟล์ **`start.bat`**
2. รอให้ระบบติดตั้ง dependencies อัตโนมัติ
3. เปิดเบราว์เซอร์ไปที่ **http://localhost:3000**

## ฟีเจอร์หลัก

### 🏠 แดชบอร์ด

- สรุปยอดขายวันนี้และเดือนนี้
- กราฟยอดขาย 7 วันย้อนหลัง
- สินค้าขายดีประจำเดือน
- แจ้งเตือนสต็อกต่ำ, ใบสั่งซื้อค้าง

### 📦 วัตถุดิบ

- จัดการวัตถุดิบ (ขวด, ฝา, ฉลาก, สารกรอง ฯลฯ)
- รับเข้า / เบิกออกสต็อก พร้อมหมายเหตุ
- ดูประวัติการเคลื่อนไหวทั้งหมด
- แจ้งเตือนเมื่อสต็อกต่ำกว่าขั้นต่ำ

### 🚰 สินค้า

- จัดการสินค้าน้ำดื่มทุกขนาด (350ml, 600ml, 1.5L, 5L, ถัง 19L)
- ตั้งราคาขาย, ต้นทุน, สต็อกขั้นต่ำ
- ติดตามสต็อกสินค้าสำเร็จรูป

### ⚙️ การผลิต

- สร้างใบสั่งผลิต
- ติดตามสถานะ (รอดำเนินการ → กำลังผลิต → เสร็จแล้ว)
- เมื่อผลิตเสร็จ ระบบอัปเดตสต็อกสินค้าอัตโนมัติ

### 👥 ลูกค้า

- จัดการข้อมูลลูกค้า 4 ประเภท (ขายปลีก, ขายส่ง, บริษัท, ตัวแทน)
- กำหนดวงเงินเครดิต
- ค้นหาและกรองตามประเภท

### 🛒 การขาย

- สร้างใบสั่งซื้อพร้อมรายการสินค้าหลายรายการ
- คำนวณยอดรวม, ส่วนลด อัตโนมัติ
- ติดตามสถานะ (รอยืนยัน → ยืนยัน → จัดส่ง → ชำระแล้ว)
- บันทึกการชำระเงิน (เงินสด, โอน, เช็ค)
- ดูรายละเอียดและประวัติการชำระต่อใบ

### 🚚 การจัดส่ง

- สร้างใบส่งของเชื่อมกับใบสั่งซื้อ
- บันทึกคนขับ, ทะเบียนรถ, วันจัดส่ง
- ติดตามสถานะการจัดส่ง
- เมื่อส่งแล้ว ระบบตัดสต็อกสินค้าอัตโนมัติ

### 📊 รายงาน

- **รายงานการขาย**: สรุปรายได้, กราฟยอดขายรายวัน, สินค้าขายดี, ลูกค้า Top 10
- **รายงานสต็อก**: มูลค่าสินค้า-วัตถุดิบ, รายการที่ต่ำกว่าขั้นต่ำ
- **รายงานการผลิต**: สรุปจำนวนผลิต, สถานะใบสั่งผลิต

### 🔐 ผู้ใช้งาน (Admin เท่านั้น)

- เพิ่ม/แก้ไขผู้ใช้งาน
- กำหนดบทบาท (Admin, Manager, Staff)
- ระงับ/เปิดใช้งานบัญชี

## โครงสร้างไฟล์

```
TWS/
├── server.js          ← Backend API (Express + SQLite)
├── package.json       ← Dependencies
├── start.bat          ← สคริปต์เริ่มต้น
├── tws.db             ← ฐานข้อมูล (สร้างอัตโนมัติ)
└── public/
    ├── index.html     ← Frontend SPA
    ├── css/style.css  ← Custom styles
    └── js/main.js     ← Frontend JavaScript
```

---

## เทคโนโลยีที่ใช้

| Layer    | Technology                            |
| -------- | ------------------------------------- |
| Backend  | Node.js + Express                     |
| Database | SQLite (better-sqlite3)               |
| Auth     | JWT + bcryptjs                        |
| Frontend | HTML5 + Bootstrap 5 + Bootstrap Icons |
| Charts   | Chart.js                              |

## การสำรองข้อมูล

ฐานข้อมูลอยู่ที่ไฟล์ `tws.db` สามารถ copy ไฟล์นี้เพื่อสำรองข้อมูลได้เลย
